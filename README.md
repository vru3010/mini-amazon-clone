# Mini Amazon Clone (Improved)

This is an improved, responsive version of a small static storefront demo.

- Pages: `index.html`, `customer-service.html`, `deal.html`, `profile.html`, `todays-deals.html`, `test_css.html`
- Styles: `style.css`, `test_css.css`
- Assets: `assets/pro_image.svg` (placeholder)

How to use
1. Open `index.html` in a browser.
2. To push to GitHub: provide the full repository URL (e.g. `https://github.com/username/repo.git`) or run the commands below locally.

Local git push (example):
```bash
cd mini-amazon-clone
git remote add origin https://github.com/USERNAME/REPO.git
git branch -M main
git push -u origin main
```
